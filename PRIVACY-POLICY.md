## 4. Data Storage and Security

### 4.1 Data Storage
- **User Data:** Stored in Supabase (PostgreSQL) hosted on AWS EU-Central-1
- **Authentication Tokens:** Stored securely on device using Expo SecureStore
- **Cache:** Temporary data stored locally on device

### 4.2 Security Measures
We implement industry-standard security measures:
- Encryption in transit (HTTPS/TLS)
- Encryption at rest (database encryption)
- Secure token storage
- Regular security audits
- Access controls and authentication

### 4.3 Data Retention
- **Active Accounts:** Data retained while account is active
- **Deleted Accounts:** Data deleted within 30 days of account deletion
- **Analytics Data:** Aggregated analytics retained for up to 2 years
- **Logs:** Server logs retained for up to 90 days
